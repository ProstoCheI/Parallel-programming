## Task №1
Представь, что у нас есть многочлен $n$-й степени:
$$P(x) = a_n x^n + a_{n-1} x^{n-1} + \dots + a_1 x + a_0$$
Если считать «в лоб», возводя $x$ в каждую степень, компьютер потратит много сил. Схема Горнера предлагает переписать это выражение, последовательно вынося $x$ за скобки:
$$P(x) = (\dots((a_n x + a_{n-1})x + a_{n-2})x + \dots + a_1)x + a_0$$
Допустим, у нас есть многочлен $2x^3 - 6x^2 + 2x - 1$ и мы хотим найти его значение при $x = 3$.
1. Берем первый коэффициент (**2**). Это наш текущий результат.
    
2. Умножаем его на $x$ (т.е. на 3) и прибавляем следующий коэффициент (**-6**):
    $2 \cdot 3 + (-6) = 0$
3. Снова умножаем результат на $x$ (на 3) и прибавляем следующий (**2**):
    $0 \cdot 3 + 2 = 2$
4. Снова умножаем на $x$ и прибавляем последний (**-1**):
    $2 \cdot 3 + (-1) = 5$

## Task №2
Основная идея — разбить многочлен на маленькие части, которые можно вычислять одновременно, а потом объединить результаты. Это превращает линейную цепочку вычислений в **дерево**.

Допустим, у нас есть многочлен 7-й степени:
$$P(x) = a_0 + a_1x + a_2x^2 + a_3x^3 + a_4x^4 + a_5x^5 + a_6x^6 + a_7x^7$$
Мы группируем слагаемые парами:
1. Вычисляем одновременно:
    $q_0 = (a_0 + a_1x)$, $q_1 = (a_2 + a_3x)$, $q_2 = (a_4 + a_5x)$, $q_3 = (a_6 + a_7x)$.
2. На следующем шаге объединяем их, используя уже вычисленное $x^2$:
    $r_0 = (q_0 + q_1x^2)$, $r_1 = (q_2 + q_3x^2)$.
3. На последнем шаге объединяем результаты, используя $x^4$:
    $P(x) = r_0 + r_1x^4$.

## Task №3
**Суть метода обратной подстановки**
Почему подстановка «обратная»? Потому что мы начинаем с самого конца, где уравнение максимально простое, и «поднимаемся» вверх.
1. **Последняя строка:** В самом низу у нас уравнение $d_{n-1}x_{n-1} = b_{n-1}$. Из него сразу находим:
$$x_{n-1} = \frac{b_{n-1}}{d_{n-1}}$$
2. **Предпоследняя строка:** Зная $x_{n-1}$, мы смотрим на строку выше: $d_{n-2}x_{n-2} + u_{n-2}x_{n-1} = b_{n-2}$. Отсюда:
$$x_{n-2} = \frac{b_{n-2} - u_{n-2}x_{n-1}}{d_{n-2}}$$
3. **Общая формула:** Для любого $i$ (двигаясь от $n-2$ до 0):
$$x_i = \frac{b_i - u_i \cdot x_{i+1}}{d_i}$$
